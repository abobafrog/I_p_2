# 🐸 Froggy Coder

**Froggy Coder** — это обучающая мини-игра на `Python + Pygame`, где игрок проходит задания по программированию, зарабатывает монеты, открывает аксессуары и сохраняет прогресс между запусками.

---

## Что есть в игре

- 2 направления обучения: **Python** и **JavaScript**.
- 3 уровня сложности: **Easy / Medium / Hard**.
- По 5 уровней в каждой ветке, по 5 заданий на уровень.
- Два типа задач:
  - выбор ответа,
  - ввод кода вручную.
- Подсказки во время прохождения.
- Система жизней (сердца) и монет.
- Магазин аксессуаров (цилиндр за монеты).
- Автосохранение прогресса в JSON.
- Адаптация интерфейса под изменение размера окна.

---

## Требования

- Python **3.10+**
- `pip`
- `pygame` **2.6+**

---

## Быстрый запуск

```bash
cd /путь/к/Game
python3 -m venv .venv
source .venv/bin/activate
pip install pygame
python game.py
```

Если команда `python` уже указывает на Python 3, можно запускать так:

```bash
python game.py
```

## Как играть

1. Запусти игру — сначала покажется splash-экран.
2. В главном меню выбери язык: **Python** или **JavaScript**.
3. Выбери сложность.
4. На карте нажми доступный уровень.
5. Решай задания:
   - для `choice` — кликай по варианту;
   - для `input` — введи ответ и нажми **«Проверить»** (или `Enter`).

### Управление

- **ЛКМ** — кнопки меню, выбор варианта, уровней и т.д.
- **Enter** — проверить ответ в задачах с вводом.
- **Esc** — пауза / продолжение во время уровня.
- **Backspace** — редактирование текста в поле ввода.

---

## Игровая логика

- За правильный ответ: **+10 монет**.
- Ошибка уменьшает количество сердец.
- Если сердца закончились, текущий уровень начинается заново.
- Подсказка доступна кнопкой **💡 Подсказка**.
- Пройденные задания/уровни сохраняются автоматически.

## Магазин и кастомизация

В разделе **«Магазин»** можно купить аксессуар:

- **Цилиндр** — 250 монет.

После покупки аксессуар активируется сразу и отображается у лягушки на разных экранах игры.

## Сохранения

Файл сохранений: `froggy_save.json`.

В нём хранится:
- монеты,
- купленные предметы,
- активный аксессуар,
- прогресс по языкам/сложностям/уровням.

### Сброс прогресса

Есть два способа:

1. Кнопка **«Сбросить прогресс»** в меню.
2. Удалить файл `froggy_save.json` вручную.

## Возможные проблемы и решения

### 1) `ModuleNotFoundError: No module named 'pygame'`
Установи зависимость:

```bash
pip install pygame
```

### 2) Игра не запускается через `python`
Используй:

```bash
python3 game.py
```

### 3) Окно открылось, но нет фонов
Проверь, что `boloto.png` и `win.jpg` лежат рядом с `game.py`.

### 4) «Сломалось» сохранение
Удали `froggy_save.json` и запусти игру снова — файл создастся заново.
